# It's Computers

**Published:** October 31, 2025

## "Everything is Different, but the Same..."

**Same:** Major business transformations haven't materialized yet. While new sectors emerged (OpenAI, Nvidia, Palantir), significant efficiency gains remain at individual levels. Tech layoffs stem more from macroeconomics than AI advancement.

**Different:** Generative AI differs from previous transformative technologies because it's general-purpose, easily integrated into existing systems, and accessible through multiple channels. Unlike domain-specific innovations like web search or social media—which required massive scale to succeed—AI components can operate independently and at various scales.

## "Things're More Moderner Than Before..."

The author critiques "AI-washing": products marketed with AI branding despite questionable technical value. Examples include high-priced "AI toothbrushes" with dubious features. The skepticism is warranted because integration of AI doesn't guarantee value. Effective AI implementation requires obvious, measurable results.

## "Everything is Bigger, and Yet, Smaller..."

**Small Integration Surface Area:** AI models integrate simply through prompt input and inference output—minimal complexity.

**Small Models:** Open-source models like Mistral-7B fit on consumer hardware. Recent innovations include models with just 7 million parameters, compared to GPT-3's 175 billion. This enables local testing with proprietary data.

## "It's Computers"

Effective AI work requires foundational computer skills:

- Linux and shell operation
- Python and Bash scripting
- Virtual environments (venv, Conda)
- Docker containers
- SSH and cloud deployment
- Git version control
- API key management

These aren't cutting-edge modern technologies—they're established operating system and deployment fundamentals. The GitHub + virtual environment + Docker architecture dominates because it solves Python dependency conflicts and enables service portability. Applications remain agnostic to whether services run locally or in the cloud.
