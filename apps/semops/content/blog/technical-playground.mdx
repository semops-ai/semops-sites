This post tests code rendering. Syntax highlighting is build-time (no client JS). Mermaid diagrams use a lightweight client component.

## Code Blocks

Here's some TypeScript with syntax highlighting:

```typescript
interface User {
  id: string;
  name: string;
  email: string;
  createdAt: Date;
}

async function fetchUser(id: string): Promise<User> {
  const response = await fetch(`/api/users/${id}`);
  if (!response.ok) {
    throw new Error('User not found');
  }
  return response.json();
}
```

And some Python:

```python
from dataclasses import dataclass
from datetime import datetime

@dataclass
class User:
    id: str
```
